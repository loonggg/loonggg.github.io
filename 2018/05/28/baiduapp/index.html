百度竟然能够做出这样的 App 来！！！

发表于 2018-05-28   |   分类于 程序人生   |   阅读次数

就在昨天的 2018 中国国际大数据产业博览会上，百度董事长兼 CEO 李彦宏发表题为《为了一个更美好的 AI 时代》演讲，在演讲中承诺对于百度搜索业务因竞价医疗广告等，“简单搜索，永远没有广告”，其中，“简单搜索” 是百度搜索在 2017 年 7 月开发和上线的 APP。

没想到百度竟然能够做出这样的 App 来！！！

昨天看到这条新闻后，我第一时间下载了这个“简单搜索”的 App ，体验试用了一番，给人的感觉确实有些惊艳，惊艳到感觉这款软件并不是百度做的。



从交互上看，确实做到了简单

这个简单搜索，操作也非常简单，基于语音搜索和文字输入搜索，当然，语音搜索放在了比较重要的位置，所以可以看出官方是建议使用语音搜索的，毕竟现在语音识别的技术已经非常成熟，另外，语音操作确实比打字速度快。

从整个界面来看，真的是非常的简约和简洁，整个首页给人的感觉非常的清爽，没有多余的界面，语音搜索或者文字输入搜索之后，直接把搜索结果展示出来，整个体验和速度都非常的好。

从搜索结果上来看，确实做到了没有广告

从搜索结果来看，我个人真的是非常的满意，我试着搜索了关键字：“非著名程序员”，而简单搜索出来的结果，排在第一的是我的 CSDN 博客，第二的是我的个人独立博客，第三的是我的 51 CTO 博客，第四个就是我的知乎专栏。整个第一页显示的结果都是正版的关于我的内容，没有显示那些山寨我文章的网站和其他第三方抄袭我文章的网站。

另外，搜索关键字：“胃病”或者”牙疼“之类的关键字，直接搜索出来的显示结果就是百度百科关于该内容的介绍等，而不是推荐一些广告或者医院。大家可以去对比一下网站端的搜索结构，真的是一目了然，反差很大。百度 Web 端，首页至少前五个都是广告，简单搜索，目前确实没有什么广告。

从角色设置来看，确实做到了人性化

这个简单搜索，虽然清新，简洁，但是 App 包的大小，还是不小，有 30 M ，其实也不多。之所以让用户感觉有点大，是因为这个简单搜索 App ，加入了角色的设置，我们可以选择原版搜索，明星搜索，和儿童搜索。

原版搜索就是极简极速，智能高效版，默认的一个搜索，而明星版其实跟原版没有什么太大的区别，只不过是有一些明星头像在你搜索的时候显示着，感觉明星陪在你什么一遍，没有什么大的用户。

重点的人性化特色就是：少儿版的搜索，如果你选择少儿版的搜索，搜索结果会过滤掉少儿不宜的内容，提供纯净的搜索结构，比如，你搜索关键字”黄色“或者”成人“，搜索结果会显示如下内容：

另外，少儿版会有百度的智能语音助手小度的陪伴，小孩可以跟小度进行有意思的对话和交流，可以听故事，看动漫，听儿歌，看手工制作等，一系列适合小孩的东西，搜索结果出来的界面也会很动漫化，非常人性化。

有人说：不出 Web 版本的话，还谈什么诚意？这个产品没有意义！还有人说：这款 App 只是用来给百度做口碑公关用的。虽然出 Web 版这样的不太现实，可能百度有做口碑的嫌疑，但是在移动互联网时代，在手机使用频率这么高的社会中，这样一款简单搜索的 App 确实也会给我们带来很多方便和好处。

百度在移动互联网时代开始掉队，而在简单搜索 App 中不仅搜索结果没有广告，而且还加入了AI 的角色，说明了百度在努力和改进。

但是无论如何，单从 App 来讲，这次百度的做法是值得肯定的，也从侧面证明了一个道理，百度用心做，还是能够做出一款非常好的软件来的。

百度从来没有承认过竞价排名是广告，而李彦宏说简单搜索永远也不会加入广告，那简单搜索在未来会不会加入竞价排名呢？希望李彦宏能够在简单搜索这款 App 中坚守住底线（对 Web 版已经不抱希望），永远不会有广告，也永远这么简洁。只有这样，可能简单搜索这款 App 才会帮住你们百度在移动互联网的搜索入口中占有一席之地，甚至扩张。
