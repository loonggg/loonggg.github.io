推荐一个不错的 Chrome 插件，百变皮肤，还可以去广告

发表于 2017-08-30   |   分类于 工具资源   |   阅读次数

今天在这里给大家推荐一个非常棒，非常不错的 Chrome 插件，功能实在是强大又好玩，让你在浏览器中可以如孙悟空一样72变，再不济，如果你不会用，不会自定义写 CSS 样式，也能够做到如猪八戒般 36 变。

今天推荐的这个插件就是：Stylish，介绍如下：

Stylish 是一个浏览器插件，适用于 Chrome，Firefox，Opera 以及 Safari 浏览器。Stylish 通过替换网页本身的 CSS 来达到的「美化」的目的。

目前，反正我在 Safari 和 Chrome 上装了此插件，但是仅仅只有在 Chrome 效果比较好，所以推荐大家在 Chrome 浏览器中使用该插件。当然，你们也可以在 Firefox 和 Opera 中试一试。
